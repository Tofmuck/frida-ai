Documents

Documentation du chantier

Notes de travail, documents de structure, hypothèses et tableaux. Une documentation vivante, pas un état définitif.

Cette page indexe la documentation publique du projet. Elle est évolutive : des documents sont ajoutés, révisés, parfois retirés. L'ordre reflète la logique du travail en cours, non une progression linéaire.

Une partie de la documentation est disponible directement ici. Une autre partie est accessible via le dépôt GitHub ↗, notamment les fichiers de structure, les schémas techniques et les expérimentations.

Hypothèses

  • H-01 — La rigueur interprétative est une propriété émergente de l'architecture, non du modèle formulée
  • H-02 — La suspension de réponse améliore la confiance à long terme plus que la réponse systématique formulée
  • H-03 — Une mémoire hiérarchisée réduit les contradictions longitudinales sans coût prohibitif en examen
  • H-04 — L'explicitation de l'incertitude modifie positivement le comportement de l'utilisateur à formuler
